## Groupe Nomblot

[Website](https://www.groupe-nomblot.com)

For this web site, I used the following technologies:
- **Frontend**
  - Framework: NuxtJS
  - CSS: TailwindCSS
- **Backend**
  - Framework: ExpressJS (and later NestJS)
  - Database: PostgreSQL
  - Messaging: RabbitMQ
- **Project management**
  - Gitlab
  - ClickUp
